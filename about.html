About Code Purple:

Our team name is Code Purple, and you may recognize us from our iconic purple color, kangaroo ears, or our famous chant:

Why Does This Website Exist?

A problem we have encountered with our new members is that many of them feel discouraged and unmotivated to join our code team.

Why is this? Well, because we code using the WPI library, which many of the aformentioned new members had never heard about.

For the past years, the to-go reponse when someone doesn't know how to code using WPILib has been, "Go read the docs."

But, especially for new coders who are interested in robotics, the documentation can be confusing and hard to understand.

Because of this, many of them eventually unfortunately give up and leave the team.

Then, most of the code ends up being done by returning members or the few who had pushed through the documentation and learned.

Due to this, the code team wishes to end this cycle and find a better way to teach.

With this project, we hope that not only new members of our team, but also new members from other teams can find an easier way to learn WPILib.
